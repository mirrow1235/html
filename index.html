Выбор базы данных для исследования

Для исследования производительности можно выбрать любую базу данных, которая соответствует требованиям исследования. Это может быть реляционная база данных, такая как MySQL, PostgreSQL или Oracle, или нереляционная база данных, такая как MongoDB или Cassandra.

Какую базу данных я выбрал?

B рамках данной курсовой работы для исследования была выбрана реляционная база данных MySQL. Это одна из самых популярных баз данных, которая широко используется в различных проектах. MySQL обладает высокой производительностью, масштабируемостью и надёжностью. Для исследования производительности будет использоваться версия MySQL Community Server.
